# Skill FitMangas — Pièges à éviter

Tirés du guide LMDM (Alex).

1. **HDR** : filmer en mode **Normal**, jamais HDR (visage délavé sur le rendu).
2. **Zones mortes Instagram** : haut ~150 px, bas ~400 px, côtés ~100 px — contenu important au centre.
3. **Coupes** : jamais laisser l’IA inventer les timestamps ; Whisper + silencedetect.
4. **Sous-titres** : pas de débordement sur la section suivante.
5. **Rendu** : HyperFrames **local** uniquement — pas de cloud HeyGen.
6. **Style** : pas de gros texte qui parrotte la voix (look IA générique).
