# Skill FitMangas — Sound design (SFX) — mapping LMDM

Source : mapping SFX du guide LMDM. Fichiers dans `assets/sfx/`.

## Principe

Claude a fabriqué les animations → il connaît les **timings exacts**.  
Poser les SFX **sur ces timings**, pas au feeling.

## Mapping (reprendre tel quel + adapter FitMangas)

| Son | Quand |
|-----|--------|
| **Riser** | Hook — finit **pile** à la dernière syllabe de l’accroche (pas « à peu près au début ») |
| **Shutter / photo** | Transitions de section — **alterner** les variantes ; **aucun** son si transition fluide |
| **Whoosh / soft swish** | Apparitions motion / cartes |
| **Feutre / draw** | Surlignages — **rogné** à la durée exacte du trait qui se dessine |
| **Clavier / type** (si dispo) | Mot-clé du CTA |
| **Pop / soft click** | CTA / logo / pill URL |

## Volumes (LMDM — calibrer à l’oreille)

- Sons **punchy** (whoosh, shutter, pop) : ~**−20 dB** sous la voix  
- Sons **fins** (feutre) : plus haut si besoin (ex. **−11 dB**) — au même volume que les punchy ils sont inaudibles  

## FitMangas

- Jamais de SFX qui masque un mot  
- Si Reel en **son d’origine seulement** (exception demandée) → **pas** de SFX / musique
